Welcome to Vantage! This guide will help you get up and running with the platform quickly.

Vantage is designed to make high-performance computing accessible through a unified interface for compute, storage, and collaboration. Whether you're running interactive notebooks or large batch workloads, the steps below walk you through the core setup required to start working productively.

## Prerequisites

Before you begin, make sure you have:

- A valid email address for account creation
- A Google or GitHub account to authenticate via SSO
- If connecting your own infrastructure: SSH access to your target machine(s)

## Quick Setup Steps

1. **[Sign Up](./sign-up.md)** - Sign up, sign in, and create a Vantage Organization
2. **[Invite team members](./invite-intro.md)** - Invite your colleagues to collaborate on projects
3. **[Connect a Cluster](./create-cluster-intro.md)** - Add compute resources to your organization
4. **[Launch a Notebook](./notebook-intro.md)** - Create and access Jupyter Notebooks in Vantage
5. **[Create a Job Script](./create-job-script-intro.md)** - Create a Job Script and define how your workload should run
6. **[Submit Your First Job](./create-job-submission-intro.md)** - Run computational workloads on your cluster

## What You Can Do

- Submit and manage computational jobs
- Scale clusters across multiple environments
- Collaborate with your team on HPC projects
- Monitor and optimize resource usage

For detailed guides, explore the specific sections in the main navigation.

### Common Use Cases

- Run batch simulations or parameter sweeps on HPC clusters
- Develop and test code interactively using Jupyter Notebooks
- Share compute resources and results with teammates
- Monitor job status, logs, and resource consumption from a single interface

Once you've completed these **Quick Setup Steps**, you'll be ready to explore the full [Platform documentation](/platform) or dive into automation with the [CLI](/cli) and [API](/api).
