# Keyboard shortcuts

| Action | Shortcut |
|---|---|
| Open command palette | <kbd>⌘</kbd> <kbd>K</kbd> |
| Search the docs | <kbd>/</kbd> |
| Jump between Workbench tabs | <kbd>g</kbd> then <kbd>s</kbd> (Sessions), <kbd>m</kbd> (Models), <kbd>e</kbd> (Endpoints), <kbd>t</kbd> (Training), <kbd>p</kbd> (Pipelines), <kbd>w</kbd> (Sweeps), <kbd>c</kbd> (Compute), <kbd>o</kbd> (Observability) |
| New session / endpoint / training job | <kbd>n</kbd> on the relevant list |
| Refresh list | <kbd>r</kbd> |
| Close detail / wizard | <kbd>Esc</kbd> |
