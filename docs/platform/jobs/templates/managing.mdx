# Managing Templates

## Actions

From a template's action menu or detail page, you can:

- **Render to Script** — open the render dialog to generate a script
- **Clone Template** — copy all template data including variables and files
- **Archive Template** — hide from default view. Preserves references from rendered scripts.
- **Restore Template** — return from archived to active state
- **Delete Template** — permanent. Removes the template and its backlink from all rendered scripts. The rendered scripts themselves are not affected.

## Identifier

You can add, change, or remove the identifier on existing templates from the detail page. Use it for quick access and filtering.

## Template Variables

You can edit variables after creation — add, remove, or change defaults.

## Files

You can add, update, or delete files from the detail page's files table.

## Filters

You can filter the template list by source (all or only mine), by whether the template has an identifier, or by archived status.
