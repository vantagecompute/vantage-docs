# Jobs

Vantage Jobs is where you define, submit, and track computational work on your clusters.

[Get started](/platform/jobs/get-started) · [Concepts](/platform/jobs/concepts)

## What you'll find inside

- **[Scripts](/platform/jobs/scripts)** — Slurm batch scripts stored in Vantage's script library. Built-in editor, support files, clone/archive/delete lifecycle. Create from scratch or render from a template.
- **[Templates](/platform/jobs/templates)** — Jinja2-based script blueprints with parameterized variables. Define a template once, let your team generate standardized scripts without touching Slurm syntax.
- **[Submissions](/platform/jobs/submissions)** — Workloads dispatched to a cluster. Real-time status tracking, resource metrics (CPU, memory, GPU, disk I/O), log viewing, and lifecycle actions (re-submit, edit, delete).
- **[Catalog](/platform/jobs/catalog)** — Pre-built job scripts that work out of the box on Vantage cloud clusters. One-click import to your script library.
